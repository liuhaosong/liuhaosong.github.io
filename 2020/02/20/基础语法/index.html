基础语法

发表于 2020-02-20 更新于 2020-03-03

打印

format

print(“我叫{}，年龄是{}”.format(name, age))%s也能像c语一样代指字符。

字符

strip

.strip()删除字符后面的空格

replace

字符代替

split

传入“，”就会以“，”为分割

join

将多个字符串化为一个字符串

列表：

空列表，含元素列表，列表加法、乘法，嵌套列表

元素：

切片，append添加，pop删除，insert插入，in|not in判断元素是否在列表，remove（）删除第一个相同元素

元组（与列表不同，不可修改）

·元素的创建

·索引（元组也可以使用索引访问元素）

·切片

·修改元组中的列表

·元组拆包

字典

可以存储任意对象，如：字符串、数字、元组、列表等。且字典中的建时唯一的，且不可变类型。包括：字典的创建

循环

·嵌套循环

·range循环

·for循环

·continue关键字

函数

·函数代码块以 def 开头，后接函数标名和括号 () 最后就是一个英文冒号 :
·传递的参数必须放在括号里面，圆括号内定义参数
·函数内容必须缩进
·函数的第一行使用文档字符串形式存放函数说明

·函数调用

·函数参数

·return关键字

·变量作用域
在所有的编程语言当中，都存在变量的作用域，一个程序的变量并不是在哪个位置都可以访问的。

·全局变量和局部变量

·global关键字
能够同时定义多个变量

·递归函数
